# 🦠 Dashboard de Análise de Covid-19

Este projeto tem como objetivo demonstrar a construção de um **dashboard interativo** utilizando **Power BI**, com dados públicos sobre a pandemia de Covid-19.

## 📌 Objetivo

Analisar o impacto da pandemia através de:
- Casos confirmados e óbitos
- Evolução temporal da pandemia
- Comparativo entre regiões
- Indicadores de vacinação e letalidade

## 🗃️ Fonte dos Dados

Os dados foram obtidos a partir de fontes públicas e confiáveis:
- [Our World in Data](https://ourworldindata.org/covid-deaths)
- [Open Data SUS](https://opendatasus.saude.gov.br/)

## 🛠️ Ferramentas Utilizadas

- Power BI
- Excel (limpeza inicial dos dados)
- Dados públicos em formato CSV

## 📊 Visualizações do Dashboard

- Série temporal de casos e mortes
- Ranking de países ou estados
- Mapas com casos por região
- Taxas de vacinação e letalidade

## 📁 Arquivos do Projeto

- `dados_covid.csv` – Conjunto de dados utilizado
- `dashboard_covid.pbix` – Arquivo do Power BI
